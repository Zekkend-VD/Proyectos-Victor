🧠 Acerca del proyecto
Este portafolio está diseñado para demostrar habilidades prácticas en JavaScript dentro de un entorno visual neón‑oscuro. Cada ejercicio se enfoca en un concepto específico de programación, manteniendo una interfaz coherente con efectos de brillo, glassmorfismo e interactividad en tiempo real.

Objetivos:

Aplicar fundamentos de JavaScript (variables, eventos, funciones, bucles, arrays, objetos).

Construir componentes responsivos y accesibles.

Combinar lógica con diseño UI/UX creativo.

Ofrecer un hub central (index) para navegar entre todos los ejercicios.

⚙️ Tecnologías utilizadas
HTML5 – Estructura semántica, canvas, formularios.

CSS3 – Flexbox, Grid, variables CSS, animaciones @keyframes, gradientes, backdrop-filter, diseño responsivo.

JavaScript (ES6+) – Funciones flecha, template literals, querySelector, listeners, métodos de array (map), objeto Math, manipulación del DOM, clases (partículas de confeti).

Font Awesome 6 – Iconos para retroalimentación visual.

Google Fonts – Tipografía Inter, limpia y moderna.

🧩 Ejercicios incluidos
#	Ejercicio	Conceptos clave	Archivo
1	Precio + IVA	Cálculo en tiempo real, parseFloat, toFixed, eventos input	ejercicio1.html
2	Multiplicación Neon	Multiplicación, inputs numéricos, actualización instantánea	ejercicio2.html
3	Dispositivos	Iteración de arrays (map), renderizado dinámico de lista, efectos hover	ejercicio3.html
4	Bienvenida + Confeti	prompt(), sistema de partículas en canvas (confeti), requestAnimationFrame	ejercicio4.html
5	Bucles Interactivos	for, while, break/continue, control deslizante, explicación de código en vivo	ejercicio5.html
6	Laboratorio de Funciones	Declaración de funciones, parámetros, retorno, registro en consola, enlace de eventos	ejercicio6.html
7	String & Number Lab	Conversión de tipos, métodos de string (toUpperCase, toLowerCase, length), Math.pow, notificaciones tipo toast	ejercicio7.html
8	Número Neon	Generación de números aleatorios, slider dinámico, valor máximo configurable	ejercicio8.html
9	Selector de Héroes	Array de objetos, renderizado dinámico de imágenes y estadísticas, cambio de fondo por categoría, evento change en <select>	ejercicio9.html
–	Portafolio Principal	Diseño en cuadrícula, tarjetas con enlaces externos, animaciones hover	index.html
Cada ejercicio es completamente independiente y está estilizado con acentos neón (cian, magenta, verde, dorado, púrpura). El index principal actúa como un tablero con nueve tarjetas – haz clic en “Ver ejercicio” para abrir la demo correspondiente en una nueva pestaña.

🛠️ Instalación y uso
Clona o descarga este repositorio.

Abre el archivo index.html en tu navegador.

Explora los ejercicios usando los botones neón.

Interactúa con cada laboratorio – modifica valores, ejecuta bucles, selecciona héroes, etc.

No necesitas servidor ni instalación – todo es front‑end puro.
